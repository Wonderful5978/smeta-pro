Установка Смета-Про

Это твоя персональная страница установки. Сохрани ссылку в закладки — отсюда же будешь качать будущие обновления. Выбирай своё устройство ниже.

📱

iPhone / iPad

iOS · через Safari

Установка на главный экран

единственный способ на iOS
  1. Открой эту страницу в Safari (не Chrome — Apple не даёт другим браузерам ставить приложения)
  2. Перейди в приложение
  3. Внизу экрана нажми кнопку «Поделиться» (квадрат со стрелкой вверх)
  4. Прокрути меню вниз → выбери «На экран Домой»
  5. Подтверди название «Смета-Про» → «Добавить»
  6. Готово — иконка появилась на рабочем столе, открывается как обычное приложение, работает оффлайн
Важно: только Safari. В Chrome на iPhone этой функции нет — это ограничение Apple, а не наше.

Android

Два варианта — выбери удобный

Вариант 1 — Установка через Chrome

рекомендуется
  1. Открой эту страницу в Chrome на телефоне
  2. Перейди в приложение
  3. Chrome предложит «Установить приложение» — нажми «Установить»
  4. Если предложения нет — открой меню (⋮ справа сверху) → «Установить приложение» / «Добавить на главный экран»
  5. Иконка появится на рабочем столе → открывается как обычное приложение
Размер: ~0,5 МБ. После первого запуска работает оффлайн.
Открыть приложение →

Вариант 2 — Скачать APK-файл

для тех, кто привык
  1. Скачай файл smeta-pro.apk
  2. Открой его в файловом менеджере → подтверди установку
  3. Android спросит: «Разрешить установку из неизвестных источников» — разреши (это обычное предупреждение для приложений не из Play Market)
  4. Готово — приложение появилось в списке
Предупреждение Android о «неизвестном источнике» — это нормально и появляется один раз. APK подписан нашим релизным ключом, обновления будут устанавливаться поверх без переустановки.
⬇️ Скачать APK (1 МБ)

Windows / Mac / Linux

Через Chrome или Edge

Установка через Chrome или Edge

единственный способ — он же самый удобный
  1. Открой приложение в браузере Chrome или Microsoft Edge (Edge уже стоит на любой Windows 10/11)
  2. В правой части адресной строки (рядом с закладками) появится маленькая иконка «Установить» — квадратик со стрелкой вниз ↓
  3. Кликни на эту иконку → во всплывающем окне жми «Установить»
  4. Готово — на рабочем столе и в меню «Пуск» появится ярлык «Смета-Про» с оранжевой иконкой «С»
  5. Запускается как обычная программа, в своём окне, без адресной строки браузера
  6. Работает оффлайн после первого открытия
Если иконки «Установить» нет в адресной строке — кликни на меню браузера (три точки ⋮ справа сверху) → в Chrome выбери «Установить Смета-Про…», в Edge → «Приложения → Установить этот сайт как приложение».
Важно: данные (профиль, цены, шаблоны) сохраняются в браузере. Не очищай данные сайта smeta-pro.pages.dev в настройках Chrome/Edge — иначе профиль пропадёт. Чтобы перенести на другой компьютер — «⚙️ Профиль → 💾 Бэкап профиля в JSON» и потом загрузить на новом устройстве.

Если ты сейчас в другом браузере (Яндекс / Opera / Firefox) — открой эту страницу в Edge или Chrome, чтобы поставить приложение:

🌐 Открыть в Microsoft Edge

Edge уже установлен на любой Windows 10/11. Если Chrome нет — скачай с google.com/chrome. Если ты уже в Chrome/Edge — просто продолжай инструкцию выше (увидишь «Установить» в адресной строке).

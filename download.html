iPhone / iPad

iOS · через Safari

Установка на главный экран

единственный способ на iOS
  1. Открой эту страницу в Safari (не Chrome — Apple не даёт другим браузерам ставить приложения)
  2. Перейди в приложение
  3. Внизу экрана нажми кнопку «Поделиться» (квадрат со стрелкой вверх)
  4. Прокрути меню вниз → выбери «На экран Домой»
  5. Подтверди название «Смета-Про» → «Добавить»
  6. Готово — иконка появилась на рабочем столе, открывается как обычное приложение, работает оффлайн
Важно: только Safari. В Chrome на iPhone этой функции нет — это ограничение Apple, а не наше.

Android

Два варианта — выбери удобный

Вариант 1 — Установка через Chrome

рекомендуется
  1. Открой эту страницу в Chrome на телефоне
  2. Перейди в приложение
  3. Chrome предложит «Установить приложение» — нажми «Установить»
  4. Если предложения нет — открой меню (⋮ справа сверху) → «Установить приложение» / «Добавить на главный экран»
  5. Иконка появится на рабочем столе → открывается как обычное приложение
Размер: ~0,5 МБ. После первого запуска работает оффлайн.
Открыть приложение →

Вариант 2 — Скачать APK-файл

для тех, кто привык
  1. Скачай файл smeta-pro.apk
  2. Открой его в файловом менеджере → подтверди установку
  3. Android спросит: «Разрешить установку из неизвестных источников» — разреши (это обычное предупреждение для приложений не из Play Market)
  4. Готово — приложение появилось в списке
Предупреждение Android о «неизвестном источнике» — это нормально. Мы не платим Google $25/год за подпись, поэтому APK не подписан их сертификатом. Файл безопасен, проверь хэш-сумму ниже.

Windows

Windows 10 · 11 · Mac · Linux

Вариант 1 — Установить как приложение через Chrome / Edge

проще всего
  1. Открой приложение в Chrome или Edge
  2. В правой части адресной строки → иконка «Установить» (квадратик со стрелкой)
  3. Нажми → подтверди → ярлык появится в меню «Пуск» и на рабочем столе
  4. Запускается как обычная программа, без браузера, работает оффлайн
Открыть приложение →

Вариант 2 — Скачать ZIP-архив

портативная версия
  1. Скачай smeta-pro-win.zip
  2. Распакуй в любую папку (например, на рабочий стол)
  3. Двойной клик на Запустить.bat — приложение откроется в Chrome как отдельное окно
  4. Можно создать ярлык на рабочем столе: ПКМ по .bat → «Создать ярлык»
Работает на любом компьютере с Chrome / Edge / другим Chromium-браузером. Все твои данные хранятся в этой папке.
